sc_add_line_tit("Title1", Colspan, Align, Background Color, Font Size, Font Color, Font Type;)

This macro must be called on a Grid onScriptInit event only.
 
The "align, background color, font size, font color, font type" parameters are optional. If left empty the Title Line settings will be used.

In align parameter you can use these options:

Parameter Description
C
Center
R
Right
L Left

 

 

 

 

*If you use this macro, you must disable Select Column option in the Toolbar.

You can add many lines as you want, just repeat the macro for each line.
