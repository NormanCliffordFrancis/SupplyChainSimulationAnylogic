sc_script_name

This macro is available to the MENU applications to access the application name that was selected in the menu. Enable to select before the application is executed if a pre-process is required to an application.


Ex. 1: To clean global variables.
if ({sc_script_name} == "apl1")
{
     sc_reset_global([global1], [global2]);
}




Ex. 2: An application requires an inexistent global variable . It is possible to call another application to process the required information, as bellow.
if ({sc_script_name} == "apl1" && (!isset([glo_company]) || empty([glo_company])))
{
     sc_redir(apl_inf_company);
}
