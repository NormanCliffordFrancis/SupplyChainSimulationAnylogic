sc_master_value('Object', Value)

This macro update a Master Application object from a Detail Application.

Parameter
 Description
Object Object name to be updated in Master Application. Its Not necessary the {} in a field name.
Value Object value. The value can be a field or a variable.
 

Ex. 1:
sc_lookup(result,"SELECT SUM(Total) FROM adm_pedido_itens WHERE PedidoID = '{PedidoID}'");
[valor] = {result}[0][0];
sc_master_value('ValorPedido', [valor]);
