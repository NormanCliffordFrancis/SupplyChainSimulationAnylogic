The event can be used to do sum values, accumulate values in variable and others functionalities.

 

Store all the fields in one variable:

 OnInit:

  [stored_variable] = "";


OnGroupBy:

  [stored_variable] .= {field_X} . " - ";


OnGroupByAll:

  [stored_variable] = explode(' - ', [stored_variable] );

if(array_search('valor_not_allowed', [stored_variable]) !== FALSE){
