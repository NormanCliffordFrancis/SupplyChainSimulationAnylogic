sc_exec_sql - This macro execute SQL commands passed as parameter or a SQL command in the SQL field action type.

sc_lookup - This macro executes a SELECT command stored in the second parameter and returns the data in a variable.

sc_select - This macro executes the commands passed in the second parameter and returns the dataset in a variable.

sc_select_field - This macro modify dynamically a field that will be recovered in the grid.

sc_select_order - This macro modify dynamically the grids "ORDER BY" clause field.

sc_select_where - This macro adds dynamically a condition to the grid WHERE clause.

sc_where_current - This macro its used to make a reference of the where clause currently used.

sc_where_orig - This macro saves the where clause content of the original application select.

variáveis database - Global variables containing database access values used by main connection.
