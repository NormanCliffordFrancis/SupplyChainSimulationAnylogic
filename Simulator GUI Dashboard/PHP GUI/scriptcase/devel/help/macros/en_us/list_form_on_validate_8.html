sc_btn_delete - This macro returns "true" when the "Delete" button is selected in a form.

sc_btn_display - This macro activate toolbars buttons on the application in execution time.

sc_btn_insert - This macro returns "true" when the "Add New" button is selected in a form.

sc_btn_new - This macro returns "true" when the "New" button is selected in a form.

sc_btn_update - This macro returns "true" when the "Save" button is selected in a form.
