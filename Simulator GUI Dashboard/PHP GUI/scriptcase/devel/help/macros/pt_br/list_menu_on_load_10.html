sc_appmenu_add_item - Esta macro adiciona um item ao menu dinamicamente.

sc_appmenu_create - Esta macro cria um menu de forma dinâmica.

sc_appmenu_exist_item - Esta Macro verifica se existe um item do menu.

sc_appmenu_remove_item - Esta macro remove dinamicamente um item do menu.

sc_appmenu_reset - Esta macro limpa o array para montagem dinâmica de um menu.

sc_appmenu_update_item - Esta macro atualiza um item do menu.

sc_menu_delete - Esta macro remove ítens da estrutura do menu.

sc_menu_disable - Esta macro desabilita ítens da estrutura do menu.
