O evento pode ser utilizado para se fazer somatórios, ou acumulo de valores, utilizando uma variável para armazenamento.

 

Armazenar todos os campo em uma variável:

OnInit:

  [variavel_de_armezenamento] = "";


OnGroupBy:

  [variavel_de_armezenamento] .= {campoX} . " - ";


OnGroupByAll:

  [variavel_de_armezenamento] = explode(' - ', [variavel_de_armezenamento] );

  if(array_search('valor_nao_permitido', [variavel_de_armezenamento]) !== FALSE){

        sc_redir('app.php', error = 'ok');

 }
