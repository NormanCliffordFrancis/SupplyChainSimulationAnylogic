Criamos um novo evento AJAX no campo cod_produto, no corpo do evento escrevemos o bloco:

 sc_lookup(vunit," select preco from produtos where codigo = {cod_produto} ");

{cpo_precp_unit_do_form} = {vunit[0][0]};
