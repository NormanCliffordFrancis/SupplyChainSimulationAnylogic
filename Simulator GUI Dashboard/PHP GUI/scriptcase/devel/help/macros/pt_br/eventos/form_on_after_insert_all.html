Este evento ocorre depois de incluir registro(s) no form de múltiplos registros.

Podemos por exemplo, após inserir os itens de uma nota fiscal, (tb_itens_nf), em um form de múltiplos registros , incluir na tabela "pai" (nota_fiscal) , o valor total dos produtos, com um pequeno bloco de código php escrito no evento  OnAfterInsertAll, veja código abaixo.

sc_lookup(tot," select sum(valor_item) from tb_itens_nf where num_nf = {num_nf} ");

{tmp_tot} = {tot[0][0]};

sc_exec_sql(" update tb_itens_nf set total_itens = {tmp_tot} where num_nf = {num_nf} ");
