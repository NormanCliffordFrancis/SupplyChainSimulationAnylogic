Este evento é executado uma única vez, antes da execução do select principal da aplicação. Neste escopo, normalmente, são executadas as macros que alteram o select, tais como: sc_select_field, sc_select_order, sc_select_where(add), etc

Podemos também carregar valor para algum atributo da aplicação , bem como verificar alguma diretiva de segurança da mesma antes de executar a aplicação
