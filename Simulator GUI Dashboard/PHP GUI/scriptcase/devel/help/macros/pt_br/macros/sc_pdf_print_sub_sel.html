sc_pdf_print_sub_sel(Array)

Imprime uma célula de um subselect. Usado internamente pelo scriptcase. Um array com parâmetros de configuração deve ser definido primeiramente.

Ex. 1:
$cell_itens_PedidoID = array('posx' => 30, 'posy' => 50, 'data' => {itens_PedidoID}, 'width' => 0, 'align' => 'L', 'font_type' => 'Times', 'font_size' => 12, 'color_r' => '0', 'color_g' => '0', 'color_b' => '0', 'font_style' => '');

$cell_itens_ProdutoID = array('posx' => 50, 'posy' => 50, 'data' => {itens_ProdutoID}, 'width' => 0, 'align' => 'L', 'font_type' => 'Times', 'font_size' => 12, 'color_r' => '0', 'color_g' => '0', 'color_b' => '0', 'font_style' => '');

$cell_itens_PrecoUnit = array('posx' => 70, 'posy' => 50, 'data' => {itens_PrecoUnit}, 'width' => 0, 'align' => 'L', 'font_type' => 'Times', 'font_size' => 12, 'color_r' => '0', 'color_g' => '0', 'color_b' => '0', 'font_style' => '');


foreach ({itens} as $NM_ind => $Dados)
{
sc_pdf_print_sub_sel($cell_itens_PedidoID[$NM_ind]);
sc_pdf_print_sub_sel($cell_itens_ProdutoID[$NM_ind]);
sc_pdf_print_sub_sel($cell_itens_PrecoUnit[$NM_ind]);
sc_pdf_ln(20);
}
