sc_seq_register

Esta macro disponibiliza o número sequencial do registro que está sendo processado na aplicação de consulta. Está disponível no evento onRecord.

 

Ex. 1:
if({sc_seq_register} == 10) {
      // ultima linha da página
      {total}  = {sum_total};
 }

 

Ex. 2:
$seq_atual = {sc_seq_register};
