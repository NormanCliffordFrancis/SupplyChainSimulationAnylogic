sc_pdf_print(Array)

 

Esta macro é utilizada internamente pelo scriptcase para criação de células. Um array com os parâmetros de configuração deve ser definido primeiramente.

 

Ex. 1:

$cell_PedidoID = array('posx' => 10, 'posy' => 10, 'data' => {PedidoID}, 'width' => 0, 'align' => 'L', 'font_type' => 'Times', 'font_size' => 12, 'color_r' => '0', 'color_g' => '0', 'color_b' => '0', 'font_style' => '');

sc_pdf_print($cell_PedidoID);
