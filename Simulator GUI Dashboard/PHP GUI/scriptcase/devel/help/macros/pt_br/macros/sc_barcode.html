sc_barcode_opção(Dados_Ent, Dados_Saida, Gerar_Tag, Altura_Img, Escala, Angulo, DPI, Tp_Img, Tp_Fonte, Tam_Fonte, CKS);

Esta macro tem como objetivo gerar códigos de barra de forma dinâmica.

Segue abaixo os tipos de código de barra junto com sua descrição:

Opção
 Descrição
sc_barcode_abar Cria o código de barras do tipo ABAR.
sc_barcode_11 Cria o código de barras do tipo CODE11.
sc_barcode_39 Cria o código de barras do tipo CODE39.
sc_barcode_39extended Cria o código de barras do tipo CODE39 Extendido.
sc_barcode_93 Cria o código de barras do tipo CODE93.
sc_barcode_128 Cria o código de barras do tipo CODE128.
sc_barcode_128a Cria o código de barras do tipo CODE128A.
sc_barcode_128b Cria o código de barras do tipo CODE128B.
sc_barcode_128c Cria o código de barras do tipo CODE128C.
sc_barcode_ean8 Cria o código de barras do tipo EAN8.
sc_barcode_ean13 Cria o código de barras do tipo EAN13.
sc_barcode_gs1128a Cria o código de barras do tipo GS1 128A.
sc_barcode_gs1128b Cria o código de barras do tipo GS1 128B.
sc_barcode_gs1128d Cria o código de barras do tipo GS1 128C.
sc_barcode_ibsn Cria o código de barras do tipo ISBN.
sc_barcode_i25 Cria o código de barras do tipo 2 de 5 Intercalado.
sc_barcode_s25 Cria o código de barras do tipo 2 de 5 Standard.
sc_barcode_msi Cria o código de barras do tipo MSI.
sc_barcode_upca 
Cria o código de barras do tipo UPCA.
sc_barcode_upce Cria o código de barras do tipo UPCE.
sc_barcode_upcext2 Cria o código de barras do tipo UPCEXT2.
sc_barcode_upcext5 Cria o código de barras do tipo UPCEXT5.
sc_barcode_postnet Cria o código de barras do tipo POSTNET.

Segue na tabela abaixo os parâmetros e sua descrição:

Parâmetro
 Descrição
Dados_Ent Variável ou string com os dados de entrada.
Dados_Saida Variável para obter os dados de saida (endereço da imagem gerada).
Gerar_Tag Gera tag html, com o src apontando para a imagem gerada. (defalt = S)
Altura_Img Altura da imagem (default 30).
Escala Fator de multiplicação da largura básica das barras (default 1).
Angulo Angulo em gráus para rotação da imagem (default 0).
DPI Resolução da imagem (default 72).
Tp_Img Tipo da imagem a ser gerada: Png, Jpeg ou Gif (default Png).
Tp_fonte Tipo da fonte (default Arial).
Tam_Fonte amanho da fonte (default 8).
CKS Se vai gerar checksun: S/N  (defalt = N) - Apenas para:
sc_barcode_39 ,  sc_barcode_39extended,  sc_barcode_i25,  sc_barcode_s25
e sc_barcode_msi.
