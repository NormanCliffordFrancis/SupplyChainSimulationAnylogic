sc_add_line_tit("Titulo1", Colspan, Align, Cor Fundo, Tamanho Fonte, Cor Fonte, Tipo Fonte;)

A macro deve ser chamada no evento onScriptInit da Consulta.
 
Os parâmetros "align, cor fundo, tam fonte, cor fonte, tipo fonte" são opcionais. Se não forem informados será assumido a configurção da linha de títulos.

No parâmetro align pode-se ser informado um dos valores abaixo:

Parâmetro Descrição
C Centralizado
R Direita
L Esquerda

 

 

 

 

*Se utilizar a macro a opção Selecionar colunas na barra de ferramentas deve está desabilitada.

Pode-se adicionar mais de uma linha, repetindo a macro para cada linha.
