sc_apl_conf - Esta macro altera as propriedades de execução das aplicações.

sc_calc_dv - Esta macro executa o cálculo de dígitos verificadores.

sc_changed - Esta macro retorna "true" se o valor do campo tiver sido modificado.

sc_decode - Esta macro retorna, o campo ou variável criptografada, à sua forma original.

sc_encode - Esta macro retorna, o campo ou variável, de forma criptografada.

sc_error_exit - Esta macro interrompe a execução da aplicação, se existir mensagens de erro geradas pela macro "sc_erro_mensagem".

sc_error_message - Esta macro gera uma mensagem de erro.

sc_exit - Esta macro força a saida da aplicação.

sc_image - Esta macro carrega, para serem usadas na aplicação, as imagens passadas como parâmentro.

sc_include - Esta macro é usada para efetuar "include" de rotinas PHP.

sc_mail_send - Esta macro é usada pra o envio de e-mails.

sc_redir - Esta macro tem por objetivo redirecionar para outra aplicação.

sc_reset_global - Esta macro elimina as váriaveis de sessão recebidas como parâmetro.

sc_set_global - Esta macro registra variáveis de sessão.

sc_site_ssl - Esta macro verifica se está sendo utilizado um site seguro. (protocolo https).

sc_trunc_num - Esta macro tem como finalidade setar o numero de casas decimais.

sc_url_exit - Esta macro altera a URL de saída da aplicação.

sc_warning - Esta macro ativa ou desativa o controle de mensagens de advertência

sc_zip_file - Esta macro é usada para gerar arquivo tipo ZIP, à partir de uma lista de arquivos e/ou diretórios.
