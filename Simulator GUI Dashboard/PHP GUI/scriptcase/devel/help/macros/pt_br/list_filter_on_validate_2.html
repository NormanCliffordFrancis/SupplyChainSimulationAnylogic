sc_date - Esta macro calcula e retorna incrementos e decrementos em datas.

sc_date_conv - Esta macro converte o conteúdo do campo passado como parâmetro do formato de entrada para o formato de saída.

sc_date_dif - Esta macro calcula a diferença entre datas em quantidade de dias.

sc_date_dif_2 - Esta macro calcula diferença entre datas, retornando a quantidade de dias, meses e anos.

sc_date_empty - Esta macro checa se um campo do tipo data está vazio, retornando um boleano.
