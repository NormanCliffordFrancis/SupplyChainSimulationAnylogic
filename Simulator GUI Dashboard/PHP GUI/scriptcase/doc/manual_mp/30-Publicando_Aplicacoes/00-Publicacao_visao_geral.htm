Deployment - OVERVIEW

It is common to test new applications in a local development environment.  Once the applications are satisfactorily tested in a secure environment then applications are released to production environment.

The deployment process involves publishing the ScriptCase developed applications (and libraries) in a PHP supported WEB server.
