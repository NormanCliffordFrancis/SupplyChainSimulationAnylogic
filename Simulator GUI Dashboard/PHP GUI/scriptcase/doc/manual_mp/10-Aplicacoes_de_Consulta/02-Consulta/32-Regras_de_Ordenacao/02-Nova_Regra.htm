Sorting Rule Settings


Image2: Sorting Rule Settings Interface.


Attributes:

  • Label - Allow to inform a name which will be shown when the application is executed.

  • Select the sorting rules fields - Allow to select through the selection bar what fields will be shown (fields which are found in the right side box) and the fields which are not shown (fields which are found in the left side box). To order the fields in the wanted way , use the ordinance bar located at the right side of the box. To apply the ordinance type to the field, you only have to select the field of the box at the right side and select the ascendant mode, where will show beside the field a + or - if the ordinance is descendant.
