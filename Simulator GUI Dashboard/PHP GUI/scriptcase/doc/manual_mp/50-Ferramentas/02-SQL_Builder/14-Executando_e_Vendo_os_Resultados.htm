RUNNING THE SQL COMMAND

Use this tab to execute the SQL command created from the selections, search and order criteria defined by user.


Figure1: Execution Folder.

Attributes>

  • SQL command : The SQL command assembled from the user information (selections, search and order criteria).
  • Records per Page - Determine number of records displayed on result. Example: 30 registers per page;
  • Run Button - Executes the created command.
  • Copy Button - Copy the SQL command to save (copied) into the application in the SQL command area.
