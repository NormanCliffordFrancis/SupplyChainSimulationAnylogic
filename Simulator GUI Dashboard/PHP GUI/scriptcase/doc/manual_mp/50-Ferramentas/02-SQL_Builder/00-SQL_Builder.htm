SQL BUILDER - Overview

SQL QUERYS building Wizard tool. Allows the choice of databases, tables, fields, joins, order by and where, visually.

To access it, the user must choose the database, the SQL command, tables and fields. The builder then assemble the SQL command. Each of the sessions below covers a portion of the building process:

  • Database Selection
  • Tables Selection
  • Fields Selection
  • Joins definition
  • Criteria Settings
  • Order Settings
  • Running the query
  • Saving the query
