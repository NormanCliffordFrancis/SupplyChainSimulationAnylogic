CSS MENUS

The CCS menu is used to create/edit styles for the menu layouts.


Figure1: ScriptCase CSS Menu editor.

 

To create a new menu theme it will be needed to choose the target of this new theme, it can be saved as Public (the theme will be avaliable to everyone who uses scriptcase to use it), it can be saved as Project (the theme will be avaliable only for this particular project) and it can be saved as User (the theme will only be avalibale to the use who create it).

On the CSS editor we can edit the CSS settings from the menu and his submenu, selecting the menu color, menu hover color, font settings, alignment and edge configuration.
