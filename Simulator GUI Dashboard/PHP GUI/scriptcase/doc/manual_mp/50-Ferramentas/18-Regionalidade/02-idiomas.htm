Languages

The ScriptCase language tool allows to translate developed application standard messages enabling a comfortable and friendly system usage.

Using this tool is possible to create customized messages to be used in different applications. A multi-language project can be developed since language selection can be changed at authentication time or during application execution.


Figure1: Language Settings.

Messages Menu

Using the messages menu is possible to select and edit messages by blocks, divided into ScriptCase Messages and My messages. New blocks can be created using the item New Folder. Selecting a block an available message index is displayed and open to edit.
