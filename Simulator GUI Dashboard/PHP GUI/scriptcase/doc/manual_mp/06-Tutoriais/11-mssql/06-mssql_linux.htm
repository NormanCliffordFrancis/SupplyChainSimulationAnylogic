Connecting with MS SQL Server on Linux

 

In this section we will see how to connect to SQLServer using ScriptCase environment preconfigured XAMPP.

Requirements::

  • Owning access to SQLServer database.
  • Have PHP configured with FreeTDS

Note: The package has FreeTDS XAMPP in your product list, but if you want to configure FreeTDS enter the site http://www.freetds.org/

4 Configuring FreeTDS

 

4.1 FreeTDS is a library that allows the connection to MS SQLServer from a Linux station.

 

Within the directory '/opt/lampp/etc' is a file named 'freetds.conf', this is the configuration file for FreeTDS.

 

4.2 Accessing the configuration file FreeTDS:

    The settings used in our case are highlighted.

        [SQLSERVER] --> It is the name that will refer to my server database.

        host --> It is the IP of my server database.

        port --> It is the port where it is installed my database.

        tds version --> It's my version of SQLServer.
