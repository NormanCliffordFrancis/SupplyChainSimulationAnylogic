Connecting to a database using Generic ODBC

Home >> Connections >> Generic ODBC

connection via "Generic ODBC" is generally used to create connections to databases dBase (.dbf), Progress, Dataflex, Paradox, Visual FoxPro, etc.. You just need to have the ODBC driver that corresponds to your database available.

This tutorial is only valid for the ScriptCase (server) running on WINDOWS.
