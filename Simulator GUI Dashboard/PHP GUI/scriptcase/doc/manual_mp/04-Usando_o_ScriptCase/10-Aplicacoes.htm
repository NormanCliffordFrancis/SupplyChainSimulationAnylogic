Applications list

Displays folders and applications. Each line offers action icons at application level.

 

1 - LABELS

Application Application Name.
Description Application Description.
Creator Application Developer.
Generation Date Generation.
Status Application Status (updated e outdated).

2 - OPTIONS

Run Run the application. If it is outdated the source code will be generated before the execution
Edit Open the application for editing.
Copy Creates an application copy. Enter a name to the copy.
Rename Changes the application name.

 

3 - Footer


It allows to run a function for multiple applications at the same time:

The checkbox allows to select all applications listed in the application list.

 

Other options are:

Generate Generate the source code.
Deploy Deploy the applications.
Delete Delete the selected applications.
Copy Copy the applications to another project.
Move Move the applications to a different folder.
Export Export the applications.
