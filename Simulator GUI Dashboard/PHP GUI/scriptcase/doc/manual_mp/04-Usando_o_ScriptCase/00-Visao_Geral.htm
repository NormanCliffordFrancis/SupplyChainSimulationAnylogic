Workspace layout Overview

The Scriptcase workspace lets you view applications, object properties and operations available. Using a integrated workspace, all options are into a single larger application window.

Once authenticated / logged in the ScriptCase, the user is redirected to a Home page similar to the "Windows Explorer". From there the user can select actions such as: create folders to organize applications, display applications previously created, copy, rename, etc.
