Link Type - Application

Used to link the Form to any other applications type. This link type is associated to submit events. When an event occurs then the application is redirected to the called application.

The “Applications List ” displayall applications. The user can choose the application that will be linked and click in Next >>


Figure4: Link between applications Interface

“Parameter definition”, at left side, display the global variablename or field name which we are linking and that needs to receive parameters. Select the option to inform the value, there are three options:

  • Field - Inform the field to be use its value.
  • Fixed - inform a fixed value.
  • Empty - No value is passed to the field.
