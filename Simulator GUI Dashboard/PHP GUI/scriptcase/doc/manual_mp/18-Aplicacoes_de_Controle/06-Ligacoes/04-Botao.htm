Link Type - Buttons

A button is created in the toolbar inside the control form, when clicked it opens the window displaying a grid or form application.

Using the opened application to search for information, add, edit or delete a record.

In “Parameter definition”, at left side, it is displayed the global variablename or field name which we are linking and that need to receive parameters for running and at the right side, will have to be selected the option to inform the value, there are two options, and they are:

  • Fixed - In this case, in the text field, you must inform a fixed value.
  • Empty - Choosing this option will not be passed any value to the field.
