SQL

 

In this group is defined the attributes relative to the connection to the database.

Figura1: SQL Settings Interface.

Attributes:

  • Connection - Show the connection name (database) that the application is using for its execution, this connection can be changed with another that have the same tables (SQL).

  • Case Sensitive - This option should be set to "No" only if the database is configured for case-sensitive and it is should not respect this setting.
