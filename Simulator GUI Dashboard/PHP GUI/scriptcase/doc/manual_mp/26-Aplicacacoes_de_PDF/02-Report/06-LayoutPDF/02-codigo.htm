Use a set of commands to manipulate the pdf report.


The Report PDF application is based on the TCPDF class. The class methods are available to the user as well macros listed in the table bellow. Use the object $pdf with the selected method, exemplo: $pdf->AcceptPageBreak(parameters);

Click here to access the TCPPF a documentation.
