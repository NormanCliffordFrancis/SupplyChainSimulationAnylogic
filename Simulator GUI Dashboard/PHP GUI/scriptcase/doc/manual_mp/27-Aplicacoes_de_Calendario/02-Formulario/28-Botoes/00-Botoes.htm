BUTTONS

Além dos botões de seus próprios aplicativos, também é possível criar botões que serão visíveis na barra de ferramentas.


Figure 1: Create new button.

 

New Button

To create a new button, click on the New Button option and enter a name and the type of the button.

Types button: (Javascript, PHP, Linking).

Deleting a button

To delete a button click the Delete icon in the right of the button name at the application menu under the Buttons option.
