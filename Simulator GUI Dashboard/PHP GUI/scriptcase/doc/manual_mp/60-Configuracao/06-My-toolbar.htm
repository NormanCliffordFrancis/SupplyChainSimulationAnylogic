My Toolbar

It allows to define your own Scriptcase toolbar.


Image1: My Toolbar Access.

 

This option is used to customize the availability and order of the toolbar items. All the menu options are available to create your customized quick bar.


Image 2: My Toolbar Interface.

This interface uses the drag and drop resource to make the arrangement of the toolbar items easier. Just move the icon to the desired position on the toolbar.

On the bottom you can find the following options:

  • Save - Save the current settings.
  • Add a separator - Add a new separator as the last item of the toolbar.
  • Create Item - Use it to create a new item on the toolbar. Iform the URL, icon and description of the new item.
  • Return to default - Restore the default settings of the Toolbar.
