Enter a backup file name and select the backup option desired:

Figure2: Partial Backup Interface.

 

Click in to select all the application elements (users, connections, projects, schemas, templates,  buttons, lookups, icons and images).

Click in the Action desired to perform the operation on selected element and select the element to include in the backup.

A final report will be displayed when backup is completed.
